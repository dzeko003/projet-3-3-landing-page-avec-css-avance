# PROJET 3.3 : Landing Page avec CSS Avancé

## Description

Ce projet a été réalisé dans le cadre de la formation **Akieni Academy – Phase 2**, **Semaine 3 : CSS - Box Model & Sélecteurs**.

L'objectif est de concevoir une **landing page moderne** inspirée des sites de produits technologiques, en mettant l'accent sur une architecture HTML sémantique et une utilisation avancée de CSS.

Ce projet permet d'approfondir les notions de cascade, de spécificité, d'héritage, de variables CSS, ainsi que l'utilisation des sélecteurs avancés et des pseudo-classes afin de produire une interface professionnelle, maintenable et accessible.

---

## Objectifs pédagogiques

- Concevoir une landing page moderne et cohérente.
- Structurer une page composée de plusieurs sections.
- Maîtriser les concepts avancés de CSS.
- Utiliser les variables CSS pour faciliter la maintenance du code.
- Comprendre la cascade, l'héritage et la spécificité.
- Produire une interface accessible et agréable à utiliser.

---

## Fonctionnalités

La landing page est composée des sections suivantes.

### Hero

- Titre principal
- Texte d'introduction
- Illustration du produit
- Bouton d'appel à l'action

### Fonctionnalités

Présentation des principales fonctionnalités du produit sous forme de cartes comportant :

- Icône
- Titre
- Description

### Témoignages

Section regroupant plusieurs avis clients mettant en avant les bénéfices du produit.

### Tarification

Comparaison de plusieurs offres comprenant :

- Nom du plan
- Fonctionnalités incluses
- Prix
- Bouton de souscription

### Appel à l'action (CTA)

Section destinée à encourager l'utilisateur à découvrir ou acheter le produit.

### Pied de page

Le footer contient les informations complémentaires ainsi que les liens utiles.

## Technologies utilisées

- HTML5
- CSS3

---

## Concepts CSS abordés

### Mise en forme

- Variables CSS (`--primary-color`, etc.)
- Cascade CSS
- Héritage des propriétés
- Spécificité des sélecteurs

### Sélecteurs avancés

- Sélecteurs descendants
- Sélecteurs enfants (`>`)
- Sélecteurs adjacents (`+`)
- Sélecteurs généraux (`~`)
- `:hover`
- `:nth-child()`
- `:first-of-type`

### Effets visuels

- Ombres (`box-shadow`)
- Bordures arrondies
- Transitions
- Animations légères
- Palette de couleurs cohérente

### Organisation

- Reset CSS personnalisé
- Code commenté
- Structure CSS claire et maintenable

---

## Critères de validation

Le projet respecte les exigences suivantes :

- Landing page composée d'au moins cinq sections distinctes.
- Utilisation des variables CSS.
- Gestion correcte de la cascade et de la spécificité.
- Utilisation des sélecteurs avancés (`>`, `+`, `~`).
- Utilisation des pseudo-classes CSS3.
- Code CSS organisé, lisible et commenté.
- Design moderne et cohérent.
- Aucun conflit de styles.
- Contraste suffisant pour garantir une bonne accessibilité.
